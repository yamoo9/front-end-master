# HTML 문서를 작성하는 기본 문법

HTML 문서를 작성하는 기본 문법을 정리해 봅니다. 웹 페이지는 헤드(head) 영역과 바디(body) 영역으로 구성됩니다.


문서 타이틀(title)은 웹 페이지의 제목으로 브라우저 탭에 표시됩니다.


# HTML 용어

자주 사용되는 HTML 용어는 다음과 같습니다.

용어 설명
element 요소
open tag 여는 태그
close tag 닫는 태그
attribute 속성
value 값

# 기본 문법

HTML 코드를 작성하는 기본 문법은 다음과 같습니다.

<태그이름 속성="값"> 컨텐츠 </태그이름>
