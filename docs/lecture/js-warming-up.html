# 워밍 업! - Console 패널

웹 브라우저 DevTool의 Console 패널은 코드를 REPL 할 수 있습니다. JavaScript 코드를 HTML 문서에 입력하는 위치 및 Console 패널 기본 사용법에 대해 학습해봅니다.

REPL

'Read Eval Print Loop'의 약자로 대화형 환경(Interactive Envrionment)을 말하며, 컴파일(Compile) 과정 없이 코드 입력과 동시에 결과를 바로 확인 할 수 있습니다.
