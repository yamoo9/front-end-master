# 💰결재 정보

보려는 페이지는 보호된 유료 콘텐츠입니다. 콘텐츠를 읽거나 시청하려면 결재가 필요합니다. 결재 방법 및 구매 패키지를 참고해 결재를 완료한 후, 패스워드를 전송 받아 패스워드를 입력하면 콘텐츠를 볼 수 있습니다. 보호된 콘텐츠를 보려면, 결재 후 전송 받은 비밀번호를 입력해주세요.

유료 콘텐츠는 사이드바 링크 텍스트 오른쪽 옆에 💰아이콘이 붙어 있습니다.

# 구매 패키지

구매 패키지는 모든 콘텐츠 또는 일부 콘텐츠로 나눠 구매할 수 있습니다. 결재된 콘텐츠는 결재 일로부터 30일간 볼 수 있습니다. '구매 후 나중에 봐야지' 하는 생각 말고, '구매 즉시 열심히 공부한다'는 마음가짐이 들었을 때 과정을 시작하세요. ➪ "선택과 집중"

패키지 가격 설명
ALL_PASS 50,000원 모든 콘텐츠를 볼 수 있습니다. (권장)
HTML_CSS_RWD_PASS 30,000원 구조/표현/반응형 디자인을 볼 수 있습니다.
JS_REST_PASS 40,000원 인터랙션/애플리케이션 디자인을 볼 수 있습니다.
