# Git GUI

Git을 보다 손쉽게 관리하기 위한 GUI Client의 종류는 운영체제 플랫폼 마다 다양하며, 무료 또는 유료로 사용할 수 있는 도구들이 있습니다. 이번 강의에서는 앞서 공부 했던 CLI 인터페이스를 보다 손쉽게 사용할 수 있도록 GUI 인터페이스 도구를 사용하는 방법을 공부해보겠습니다.
