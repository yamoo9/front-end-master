# 인용문

문장 내에서 인용 구문이 사용될 필요가 있다면 <q> 요소로 마크업 합니다. (큰 따옴표("), 작은 따옴표(')로 대체 가능) 여러 문장을 묶은 인용 구문이 필요하다면 <blockquote> 요소로 구조화 합니다. 출처는 <cite> 요소를 사용하세요.

# Q & A

수강생과 주고 받은 질문/답변 입니다. 참고하세요.

Q1

1. cite 태그로 쓸 경우 폰트가 이탤릭처럼 보여지나요?
2. cite 태그로 썼을 때 보면 처럼 문단이 바뀌는데 블록 요소라고 생각하면 되나요?
3. cite 태그 = 출처를 남길 때 만 단독으로 써도 되나요? ul li처럼 와 항상 같이 쓰여져야 하나요?
4. q를 쓰면 " 로 표시되는데.. 상황에 따라서 " 와 ' 나올 수 있게 가능한가요 ??

A1

1. 네 맞습니다. 하지만 시각적인 모양은 CSS로 변경 가능합니다.
2. 아닙니다. cite 요소는 인라인 요소입니다.
3. cite 요소는 구문 컨텐츠를 허용하는 모든 요소를 부모로 할 수 있습니다.
4. 반드시 q 요소를 사용할 필요는 없습니다. 작은 따옴표('), 큰 따옴표(") 기호를 그대로 사용하셔도 됩니다.
    (참고로 인용 부호 모양은 CSS로 사용자가 임의로 조작할 수 있습니다.)
