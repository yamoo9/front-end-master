# JavaScript란?

웹 애플리케이션(또는 사이트) 개발에 있어 구조는 HTML을 사용해서 구축하고, 스타일링은 CSS를 사용해 표현하죠. 그렇다면 웹의 특성인 인터랙션(Interaction: 상호작용)은 어떻게 구현해야 할까요? 네. JavaScript 가 그러한 역할을 수행합니다. 아래 뮤직 플레이어 곡을 선택한 후, 플레이(▷)을 누르면 음악이 흘러 나옵니다. 이러한 기능 구현을 JavaScript로 만들게 됩니다. 재미있겠죠? 😃
