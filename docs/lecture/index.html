# 강의 소개

이듬(EUID)'s 'Front-End Master' 과정은 기본적으로 블렌디드 러닝 방법으로 진행됩니다. 블렌디드 러닝 모집 공고를 통해 과정을 신청할 수 있습니다.

Chrome 웹 브라우저를 사용해 Front-End Master 강의 서비스를 이용하길 권장합니다. (IE, Edge, Firefox 미지원)

온라인 강의만 듣고 싶다면?

블렌디드 러닝 스타일 대신, 온라인 강의만 학습하는 것도 가능합니다.
강의 중 일부는 무료로 볼 수 있지만, 유료 콘텐츠의 경우는 결제 후에 볼 수 있습니다.

# 강의 커리큘럼

좌측 사이드 바의 각 카테고리를 통해 커리큘럼 콘텐츠를 살펴볼 수 있습니다. 참고로 좌측에서 안내된 강의 콘텐츠는 온라인 전용입니다. 블렌디드 러닝 강의는 기본적으로 온라인 강의에 더해 오프라인 강의 콘텐츠를 보다 심도 있게 다루게 됩니다.

온라인 강의 콘테츠는 총 33시간 분량이며, 127개의 동영상을 제공합니다.

스텝 카테고리 설명
00 강의 시작에 앞서 학습 내용 및 준비 사항 소개
01 구조 디자인 (HTML) 웹 문서 골격을 구성하는 HTML 표준 컴포넌트 사용법에 대해 학습
02 표현 디자인 (CSS) 웹 문서를 스타일링 하는 CSS 표준 사용법에 대해 학습
03 반응형 디자인 (RWD) N 스크린(모바일, 데스크탑 등 다양한 화면)에 대응하는 반응형 웹 디자인 방법에 대해 학습
04 인터랙션 디자인 (JavaScript) 사용자와의 인터랙션을 통해 웹 서비스를 구동하는 방법에 대해 학습
05 인터랙션 디자인 (DOM) DOM API를 사용해 문서를 컨트롤 하는 방법에 대해 학습
06 비동기 프로그래밍 (AJAX) 서버와의 비동기 통신을 통해 서비스를 처리하는 방법에 대해 학습
07 RESTful 웹 서비스 (API) REST API를 사용해 서버 개발자와 통신하여 CRUD 하는 방법 학습
08 모던 자바스크립트 (ES6+) 모던 JavaScript를 사용해 프로그래밍 하는 방법 학습 (React, Vue 등에서 활용 됨)
09 SVG 드로잉 + 애니메이션 SVG 드로잉, 애니메이션 하는 방법 학습
10 개발 도구 (NodeJS) 워크플로우 자동화 및 모던 JavaScript를 바로 사용하기 위한 도구 사용법 학습
11 버전 관리 도구 (Git) Git, GitHub, Git GUI를 사용하는 방법 학습

# 블렌디드 러닝

블랜디드 러닝(Blended Learning)은 2가지 이상의 학습 방법을 결합한 학습 방법을 말합니다. 일반적으로는 온라인 학습과 오프라인 학습이 합쳐진 학습 방법을 말합니다. 2010년 미국 교육부에서 실시한 블랜디드 러닝의 메타 분석에 따르면 100% 오프라인 또는 온라인 강의 보다 효과적인 것으로 나타났습니다.
