# 💸 결재 정보

본 서비스의 일부 페이지는 유료 콘텐츠로 보호되고 있어,
일부 유료 콘텐츠 서비스를 이용하려면 먼저 회원 가입 절차가 필요합니다.

# STEP 1

회원 가입은 상단 메뉴바 왼쪽에 위치한 로그인 버튼을 눌러 로그인 페이지로 이동한 후, 로그인 폼 하단의 ← 회원 가입 페이지 링크를 클릭해 진행합니다. 현재 가입 방법은 이메일 또는 Google 계정으로만 가입이 가능합니다. 가입이 완료되면 '가입 완료' 됨을 알리며 자동으로 로그인 사용자 페이지로 이동됩니다.

# STEP 2

로그인 후 사용자 페이지를 통해 현재 접근 가능한 수강 권한 정보를 확인할 수 있습니다. 기본 권한은 FREE_USER 입니다. FREE_USER는 무료 강의에 한해 제한 없이 강좌를 시청할 수 있습니다.

유료 콘텐츠는 사이드바(왼쪽) 링크 텍스트 오른쪽 옆에 💰 아이콘이 붙어 있습니다.

강좌를 구매한 후에는 사용자 페이지 정보가 변경됩니다. (아래 이미지 참고)

# 구매 패키지

구매 패키지는 모든 콘텐츠 또는 일부 콘텐츠로 나눠 구매할 수 있습니다. 결재된 콘텐츠는 구매 일로부터 30일간 볼 수 있습니다. '구매 후 나중에 봐야지' 하는 생각 말고, '구매 즉시 열심히 공부한다'는 마음가짐이 들었을 때 과정을 시작하세요. ➪ "30일 간의 선택과 집중"

평생 볼 수 있는 구매한 도서, 구매한 유료 강좌 잘 챙겨보시나요?
구매하고 나서 안 볼 바에는, 제한 된 시간 안에 확고한 목표를 가지고 보는 것이 x100 낫습니다.

배지 패키지 가격 설명
🥇 ALL_PASS 50,000원 제약 없이 모든 콘텐츠를 볼 수 있습니다. (권장)
🥈 HTML_CSS_RWD_PASS 30,000원 구조 / 표현 / 반응형 디자인 콘텐츠를 볼 수 있습니다.
🥉 JS_ES6_REST_PASS 40,000원 인터랙션 디자인 이후 모든 콘텐츠를 볼 수 있습니다.
