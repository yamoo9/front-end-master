# 💸 결재 정보

본 서비스의 일부 페이지는 유료 콘텐츠로 보호되고 있어, 일부 유료 콘텐츠 서비스를 이용하려면 먼저 회원 가입 절차가 필요합니다.

유료 콘텐츠는 사이드바 링크 텍스트 오른쪽 옆에 💰아이콘이 붙어 있습니다.

# STEP 1

회원 가입은 상단 메뉴바 왼쪽에 위치한 로그인 버튼을 눌러 로그인 페이지로 이동한 후, 로그인 폼 하단의 '가입' 링크를 클릭해 진행합니다. 가입이 마무리 되면 가입한 이메일 주소로 유효한 이메일인지 컨펌(Confirm) 메일이 발송됩니다. 메일 확인 후 승인 링크를 클릭해 계정을 활성화 합니다.

📩

# STEP 2

계정이 활성화 되면 결재 방법 및 구매 패키지 정보를 참고한 후, 결재  ➪  승인 과정을 거쳐 구매한 콘텐츠를 볼 수 있습니다. 구매 후 서비스에 로그인 하면 사용자 이름 왼쪽 옆에 구매 콘텐츠 배지가 표시됩니다. (아래 이미지 참고)

# 구매 패키지

구매 패키지는 모든 콘텐츠 또는 일부 콘텐츠로 나눠 구매할 수 있습니다. 결재된 콘텐츠는 구매 일로부터 30일간 볼 수 있습니다. '구매 후 나중에 봐야지' 하는 생각 말고, '구매 즉시 열심히 공부한다'는 마음가짐이 들었을 때 과정을 시작하세요. ➪ "30일 간의 선택과 집중"

배지 패키지 가격 설명
🥇 ALL_PASS 50,000원 제약 없이 모든 콘텐츠를 볼 수 있습니다. (권장)
🥈 HTML_CSS_RWD_PASS 30,000원 구조 / 표현 / 반응형 디자인 콘텐츠를 볼 수 있습니다.
🥉 JS_REST_PASS 40,000원 인터랙션 디자인 이후 모든 콘텐츠를 볼 수 있습니다.
