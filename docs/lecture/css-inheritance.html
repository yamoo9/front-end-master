# 상속 (Inheritance)

CSS 속성 중 일부는 상속이 되고, 일부는 그렇지 않은 속성도 있습니다.
상속에 대한 개념과 상속이 되는 속성, 그렇지 않은 속성에 대해 학습해 봅니다.
