# IIFE → Block

IIFE 패턴은 일반적으로 변수들을 별도의 영역 안에서만 사용하기 위해 사용되었습니다. ES6+에서는 Block을 기반으로 영역을 만들 수 있으므로, 더 이상 함수 기반 영역을 사용하지 않아도 됩니다.

IIFE 패턴을 사용하는 경우





 

(function () {
  var food = '허니버터칩';
}());

console.log(food); // Reference Error

ES6+ 블록을 사용하는 경우





 

{
  let food = '허니버터칩';
};

console.log(food); // Reference Error
