# 최고의 JavaScript 교과서는?

살아있는 온라인 참고서(Reference)야 말로 최고의 교과서라 할 수 있을 겁니다. 종이 책을 무시하는 것은 아니지만, 한 번 저술된 도서는 기술의 변화를 발 빠르게 업데이트 하지 못합니다. 반면 온라인 참고서는 늘 새로운 기술을 반영하여 콘텐츠가 업데이트 되죠! 더 이상의 교과서가 있을까요? ^^
