# CSS 기본 문법

CSS 언어를 작성하는 기본 문법은 다음과 같습니다.

선택자 {
  속성1: 값;
  속성2: 값2;
}

# 스타일 방법 3가지

CSS 코드를 HTML 문서에 적용하여 스타일링하는 3가지 방법을 학습해봅니다.

  • 인라인 스타일 (inline style)
  • 인터널 스타일 (internal style)
  • 익스터널 스타일 (external style)
