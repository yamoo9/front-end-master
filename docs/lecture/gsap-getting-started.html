# GSAP 소개

GSAP(Green Sock Animation Platform)에 대해 소개합니다.

극강의 퍼포먼스, 프로페셔널 모던 웹 애니메이션 라이브러리

GSAP은 대부분의 모던 브라우저에서 작동하는 고성능 애니메이션 JavaScript 라이브러리 입니다. 고급 시퀀싱(sequencing), 안정성, API 효율성(efficiency) 및 엄격한 제어(tight control) 기능을 제공하는 동시에 3백만 개가 넘는 사이트의 실제 애니메이터 문제를 해결합니다. GSAP은 CSS 속성 뿐만 아니라, 모든 JavaScript 객체의 모든 숫자 속성에 애니메이션을 적용 할 수 있습니다.

GSAP을 애니메이션 맥가이버 칼(Swiss Army Knife)로 생각하세요. GSAP은 CSS 속성은 물론 Canvas 라이브러리 객체, SVG, JavaScript 객체 등 JavaScript로 조정 가능한 모든 것을 애니메이션할 수 있으며 브라우저 호환성 문제를 해결합니다.

게다가 트랜스폼(transforms) 애니메이션 시, 자동으로 GPU 가속화(acceleration)가 활성화되어 놀라운 속도를 선보입니다. (jQuery 보다 20배 이상 빠름) 왜? GSAP을 사용하면 좋은지 자세히 알고 싶다면 "Why GSAP?" 문서 내용을 읽어보세요.

대부분의 라이브러리가 CSS 속성에 국한하여 애니메이션 할 수 있는데 반해 GSAP은 시퀀싱, 런타임 컨트롤, 플러그인 등 다양한 방법을 제공하므로 다소 과정해서 말하면 지구상에서 가장 강력한 고성능 애니메이션 라이브러리라고 부를 수 있을 겁니다. 뿐만 아니라 GSAP은 모놀리(monolithic, 획일적이고 자유성 결여)식 프레임워크와 달리 독립적이고 유연하게 사용할 수 있습니다.

GSAP, Draggable, TweenLite, TweenMax, TimelineLite, TimelineMax, 다양한 플러그인, 이징(easing) 함수 등 모든 애니메이션 도구를 제공합니다.
