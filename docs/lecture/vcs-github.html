# GitHub

Git에 대한 공부를 마쳤으니 최종적으로 여러분의 프로젝트를 관리할 온라인 서비스가 필요합니다. 프로젝트 관리 목적으로 사용되는 대표적인 온라인 서비스로 github.com을 들 수 있습니다. 자! 그럼 프로젝트를 관리 온라인 서비스 중 GitHub을 공부해봅시다.
